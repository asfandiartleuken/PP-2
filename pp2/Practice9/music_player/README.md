# Music Player (Practice 9)

Keyboard-controlled music player using `pygame.mixer.music`.

## Controls
- `P` тАФ Play / resume
- `S` тАФ Stop
- `N` тАФ Next track
- `B` тАФ Previous track
- `SPACE` тАФ Pause / Resume
- `Q` or `Esc` тАФ Quit

## Running
```
python main.py
```
Ensure `pygame` is installed (see repo-level `requirements.txt`).
