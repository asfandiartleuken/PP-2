# Mickey's Clock (Practice 9)

Analog-style clock that uses Mickey Mouse hands for the minute and second hands. Syncs to system time.

## Features
- Right hand = minutes, left hand = seconds
- Updates in real time using system clock
- Digital time readout and legend
- Works with bundled vector-drawn hands (no external image required)

## Controls
- Q or Esc: quit

## Running
```
python main.py
```
Ensure `pygame` is installed (see repo-level `requirements.txt`).
