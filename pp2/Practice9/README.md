# Practice 09: Pygame Mini Projects (Part 1)

This folder implements three small Pygame exercises:
- Mickey's Clock (`mickeys_clock/`)
- Keyboard Music Player (`music_player/`)
- Moving Ball (`moving_ball/`)

## Setup
Install dependencies (only `pygame` needed):
```
pip install -r requirements.txt
```

## Project notes
- Each subfolder has its own `main.py` and README with controls.
- Sample WAV tracks are generated into `music_player/music/sample_tracks/` on first run via provided script.
- Virtual environment files under `venv/` are optional; you can ignore if using your own env.

## Run examples
```
python mickeys_clock/main.py
python music_player/main.py
python moving_ball/main.py
```
